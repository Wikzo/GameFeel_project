Investigating Game Feel - Aalborg University, Medialogy (2015)

Thanks for taking your time to participate in this test! As a token of appreciation, by completing the test (takes 10-15 minutes), you will enter a contest where one randomly-chosen participant wins a $10 gift card for Amazon or Steam.

You will be presented with a 2D platforming game where you have to collect 3 stars. You will play the same game 4 times, but each time some of the parameters change. Between the rounds, you will be asked to answer some simple questions. These questions are about how the controls feel.

You have two options (all versions are identical):

Option 1: Web Version (requires Unity plugin)

Runs directly in your browser using the Unity plugin. Please note that Google Chrome from version 42 have disabled the Unity plugin per default. To enable the plugin again, type chrome://flags/#enable-npapi into the address bar and click on Enable NPAPI and restart the browser.

Option 3: Standalone Download (Windows)

Download executable to your computer. Don't worry if Google Chrome warns you about the file being dangerous. Just download the .zip file and extract it (e.g., by using 7-Zip or similar).
